## 推荐系统定义

推荐系统
定义：根据用户的历史信息和行为，向用户推荐他感兴趣的内容

基于行为的协同过滤
基于内容的相似推荐

推荐系统解决的问题

1. 信息过载
2. 挖掘长尾
3. 用户体验

## 推荐系统环节

低延迟 50ms\~300ms

推荐系统环节：

```
      输入 -> 召回 -> 排序 -> 调整 -> 输出
数据量 百万     万             千     百
```

1. 召回：

- 协同过滤召回
- 内容相似召回
- 图算法召回
- 热门召回
- 新课召回
  ...

2. 排序

- 机器学习-二分类算法
  - LR
  - GBDT
  - DNN
  - Wide&Deep

3. 调整

- 去重
- 已购/已读过滤
- 在线过滤
- 热门补足
- 分页提取
- 合并内容信息
