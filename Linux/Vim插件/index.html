    Vim插件

    vimscript语言编写功能扩展
    插件管理器
    vim-plug vundle pathogen dein.vim volt

    推荐 vim-plug
    https://github.com/junegunn/vim-plug

    安装开屏插件
    vim-startify
    https://github.com/mhinz/vim-startify

    修改~/.vimrc

    call plug#begin('~/.vim/plugged')
Plug 'mhinz/vim-startify'
call plug#end()

    保存:w
    重启vim或者 source ~/.vimrc
    执行:PlugInstall
    重新打开vim就可以看到效果
    :qa # 退出所有窗口
